Tentativa 1

A quinta tentativa foi com dados de janeiro de 2000 a outubro de 2017 incluiu as seguintes vari√°veis:

  1. Raz√£o capital-trabalho, calculada a partir das seguintes s√©ries:
    • Receitas tribut√°rias - Regime de compet√™ncia - Imposto de renda - Retido na fonte - Rendimento do trabalho (7620)
    • Receitas tribut√°rias - Regime de compet√™ncia - Imposto de renda - Retido na fonte - Rendimento do capital (7621)

Passei um filtro para tirar sazonalidade, sem transforma√ß√£o pr√©via nos dados (ajuste default na fun√ß√£o seas exceto no transformation que ficou igual a NONE).

  1. Taxa Selic calculada a partir da s√©rie:
    • Taxa de juros - Selic acumulada no m√™s (4390). Passei de taxa mensal para anual usando a f√≥rmula: \(\left((1+tx/100)^12 -1\right)*100\) e passei um filtro para tirar sazonalidade, sem transforma√ß√£o pr√©via nos dados (ajuste default na fun√ß√£o seas exceto no transformation que ficou igual a NONE).
  2. IPCA (para infla√ß√£o), utilizando a seguinte s√©rie:
    • √çndice nacional de pre√ßos ao consumidor-amplo (IPCA) (433). Como ele est√° em varia√ß√£o percentual mensal, foi necess√°rio transformar para acumulado dos √∫ltimos 12 meses utilizando a f√≥rmula: \[IPCA_i = \left[\left(\prod\limits_{j=i-11}^i \left(\frac{IPCA_j}{100}+1\right) \right) -1\right]*100\]
  3. PIB mensal, utilizando a seguinte s√©rie:
    • PIB mensal do IPEA Data. Eu deflacionei no pr√≥prio site usando como deflator o IPCA de novembro de 2017. Depois tirei o log e calculei a primeira diferen√ßa e tirei sazonalidade.
  4. Crescimento da taxa de c√¢mbio (para growth of the nominal effective exchange rate). Usei a s√©rie:
    • Taxa de c√¢mbio - Livre - D√≥lar americano (venda) - Fim de per√≠odo - mensal (3696). Como o Mumtaz disse que usou log diferen√ßa, eu calculei o log e tirei primeira diferen√ßa nessa s√©rie.

desc <- function(x) {
  n       <- length(x)
  minimum <- min(x, na.rm = TRUE)
  first_q <- quantile(x, 0.25, na.rm = TRUE)
  media   <- mean(x, na.rm = TRUE)
  mediana <- median(x, na.rm = TRUE)
  third_q <- quantile(x, 0.75, na.rm = TRUE)
  maximum <- max(x, na.rm = TRUE)
  std     <- sd(x, na.rm = TRUE)

    return(list(n = n, minimum = minimum, first_quar = first_q, media = media, mediana = mediana, third_quar = third_q, maximum = maximum, std = std))
